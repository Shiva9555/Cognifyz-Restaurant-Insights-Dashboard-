# Cognifyz Restaurant Insights Dashboard 📊

![Project Badge](https://img.shields.io/badge/Cognifyz-Data%20Analysis%20Internship-indigo)
![License](https://img.shields.io/badge/License-MIT-green)

A premium, interactive data analysis dashboard built for the **Cognifyz Technologies Data Analysis Internship**. This project explores a global restaurant dataset across three levels of complexity, providing deep insights into cuisine trends, pricing strategies, and consumer behavior.

---

## 🚀 Live Demo & Visuals
The dashboard features a **Premium Glassmorphism UI** with interactive charts and maps. 

*   **Interactive Tabs**: Navigate between Fundamentals (Level 1), Intermediate (Level 2), and Advanced (Level 3) insights.
*   **Global Map**: Explore restaurant clusters using interactive geographic visualization.
*   **Predictive Engine**: A built-in logic to understand rating correlations.

---

## 📈 Internship Tasks & Completion

### Level 1: Fundamentals
*   **Task 1: Top Cuisines**: Identified the top 3 most common cuisines and calculated their market share percentage.
*   **Task 2: City Analysis**: Identified the city with the highest density of restaurants and the city with the highest average rating.
*   **Task 3: Price Range Distribution**: Visualized the frequency of price ranges across the global dataset.
*   **Task 4: Online Delivery**: Analyzed the availability of online delivery and its direct impact on restaurant ratings.

### Level 2: Intermediate
*   **Task 1: Restaurant Ratings**: Analyzed the distribution of aggregate ratings and determined the most common rating ranges.
*   **Task 2: Cuisine Combinations**: Identified the most popular multi-cuisine pairings and their performance.
*   **Task 3: Geographic Analysis**: Mapped restaurant locations globally using Latitude/Longitude to identify spatial clusters.
*   **Task 4: Restaurant Chains**: Isolated restaurant chains and analyzed their outlet counts vs. ratings.

### Level 3: Advanced
*   **Task 1: Restaurant Reviews**: Performed keyword analysis on review texts and explored the relationship between review length and rating.
*   **Task 2: Votes Analysis**: Identified restaurants with extreme voting patterns and analyzed the correlation between votes and star ratings.
*   **Task 3: Price Range vs. Services**: Demonstrated how pricing influences the availability of table bookings and online delivery.

## 🛠️ Technology Stack
*   **Frontend**: HTML5, Vanilla CSS (Glassmorphism), JavaScript (ES6)
*   **Data Processing**: PapaParse (CSV Parsing)
*   **Visualization**: Chart.js v4, Leaflet.js (Mapping)
*   **Analysis Logic**: Python (Core analysis scripts)

## ⚙️ How to Run Locally
1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/your-username/cognifyz-restaurant-dashboard.git
    ```
2.  **Open the Project**:
    Navigate to the project folder and open `index.html` in any modern web browser (Google Chrome or Brave recommended for best performance).
3.  **Data Loading**:
    The dashboard automatically fetches data from `dataset.csv`. Ensure the CSV file is in the same directory as `index.html`.
